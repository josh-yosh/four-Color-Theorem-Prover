1. Rotation Systems CCW Angle Sort

At each vertex, we sort its neighbors counter-clockwise by angle using atan2. This cyclic ordering is called a rotation system — it's the only information we need to find all faces.

angle(neighbor) = atan2(neighbor.y − v.y, neighbor.x − v.x)
sort neighbors ascending → CCW order

2. Half-Edge Rotation Rule Face Tracing

Each undirected edge splits into two directed half-edges. The rule to move to the next half-edge in the same face: arrive at vertex v via u→v, then look up u in v's CCW list and step one position clockwise (= one step backward) to find the next neighbor w. Follow v→w. Repeat until you return to the start — that loop is one face.

φ(u→v) = v→w, where w = CCW_neighbors[v][ (index_of_u − 1) mod degree(v) ]

3. Shoelace Formula Signed Area

After tracing a face we get an ordered list of points. The shoelace formula gives a signed area — positive means CCW (interior face ✅), negative means CW (outer face ❌).

A = ½ · Σ (xᵢ·yᵢ₊₁ − xᵢ₊₁·yᵢ) for i = 0…n−1 (mod n)

A > 0 → CCW winding → interior face ✅
A < 0 → CW winding → outer face ❌

4. Euler's Formula Sanity Check

After running the algorithm you can verify all faces were found correctly:

V − E + F = 2 (F includes the 1 outer unbounded face)

∴ interior faces = E − V + 1
